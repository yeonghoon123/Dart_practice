## 2. Dart란?

구글에서 개발한 프로그래밍 언어로, 주로 모바일 앱 및 웹 애플레케이션을 개발하기 위해 사용된다.<br>
객체 지향적이며, 정적 타입 언어로서 개발자가 코드를 더 안정적으로 유지하고 디버그하기 쉽게 만들어준다.<br>
가비지 컬렉션을 지원하며, C 언어와 비슷한 문법을 가지고 있어 다른 언어에서 이미 경험이 있는 개발자들이 쉽게 학습할 수 있다.<br>
